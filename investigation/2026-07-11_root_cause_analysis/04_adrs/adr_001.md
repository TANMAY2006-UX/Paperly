# ADR-001 — Semantic Candidate Selection
*Engineering Decision Record for Architecture Evolution*

---

## Part 1 — Proposed Architectural Modifications

Throughout the investigation, the following structural changes have been proposed:
1.  **Evaluate Set B Directly (Remove `is_outlier` Gate)**: Allow `semantic.py` to evaluate any typographic group that is not part of the universal body class, regardless of whether Landmark flagged it as an outlier.
2.  **Incorporate Spatial Geometry into Semantic**: Inject bounding box intersection logic into Semantic to resolve Front Matter ambiguities.
3.  **Redefine Landmark Contract**: Downgrade Landmark from a global structural candidate generator to a specialized Anchor/Endpoint detector.
4.  **Refine Lexical Regular Expressions**: Expand the regex logic in Semantic to cover authors and unnumbered titles.

---

## Part 2 — Evidence Evaluation

### Proposal 1: Evaluate Set B Directly
*   **Problem solved:** Semantic is currently blind to 70% of structural objects because Landmark's mathematical model drops them.
*   **Proved by:** EXP-06 (Candidate Enumeration proved `is_outlier` only captures 29.65% of structural data).
*   **Supported by:** EXP-08 (Lexical regex perfectly recovers Set B headers), EXP-09 (Parser State isolates Set B noise).
*   **Contradicted by:** None.
*   **Assumptions remaining:** Does the Universal Body Class (UBC) act as a strong enough boundary to prevent the massive Set C (body paragraphs) from overwhelming the parser?

### Proposal 2: Incorporate Spatial Geometry into Semantic
*   **Problem solved:** Authors and unnumbered titles cannot be deterministically identified using text or physical properties alone.
*   **Proved by:** EXP-07 (Physical overlap), EXP-08 (Lexical overlap), EXP-09 (Front Matter state overlap).
*   **Supported by:** Architecture Review 001 (Spatial is the only dimension remaining).
*   **Contradicted by:** None.
*   **Assumptions remaining:** We have not experimentally proven that bounding boxes perfectly resolve this specific ambiguity.

### Proposal 3: Redefine Landmark Contract
*   **Problem solved:** Landmark is fundamentally mathematically incapable of detecting valleys (smaller font headers) using local maxima.
*   **Proved by:** PIP-003, EXP-01.
*   **Supported by:** EXP-06 (Only 29% coverage).
*   **Contradicted by:** None.
*   **Assumptions remaining:** None.

### Proposal 4: Refine Lexical Regular Expressions
*   **Problem solved:** Authors and Titles do not match existing regexes.
*   **Proved by:** EXP-08 (70% of Set B objects lack regex signatures).
*   **Supported by:** None.
*   **Contradicted by:** EXP-08 (Authors and unnumbered titles are lexically identical to table values and inline math. Writing a regex for "John Doe" will identically match "Max Temp").
*   **Assumptions remaining:** None.

---

## Part 3 — Classification

*   **Proposal 1 (Evaluate Set B Directly)**: **A. Evidence sufficient for implementation**. We know exactly what information gain we get (30% recovery of headers in the Body) and we know exactly how to do it.
*   **Proposal 2 (Incorporate Spatial Geometry)**: **B. Promising but requires another experiment**. We know geometry is required, but we have no telemetry on it yet.
*   **Proposal 3 (Redefine Landmark Contract)**: **A. Evidence sufficient for implementation**. This naturally flows from Proposal 1. Semantic will stop trusting `is_outlier` but will continue to trust `is_anchor`.
*   **Proposal 4 (Refine Lexical Regular Expressions)**: **C. Rejected**. Lexical separation for Authors is mathematically impossible without catastrophic false positives.

## Part 5 — Minimal Implementation Roadmap

*Every step modifies exactly one architectural responsibility and is independently testable.*

**Step 1: Relax Semantic Gate**
Modify `semantic.py` to allow any group into the evaluation loop if `group.typography_class != universal_body_class`. 

**Step 2: Apply Existing Regex**
Within `BODY_PARSE`, allow `SECTION_REGEX` and `SUBSECTION_REGEX` to validate these newly admitted Set B candidates.

**Step 3: Geometry Experiment**
Design an experiment to measure spatial intersections in the Front Matter to solve the Author ambiguity.

## Part 6 — Implementation Milestones

### Milestone 1: Body Parse Structural Recovery
*   **Objective:** Recover the missing section headers (PIP-003/EXP-08) by removing the `is_outlier` gate in `semantic.py` for body items.
*   **Code Change:** In `reconstruct_semantics()`, change the check from `if is_outlier:` to `if is_outlier or group.typography_class != universal_body_class:`.
*   **Benchmark:** Re-run Spanner parsing. 
*   **Success Criterion:** Spanner headers "1. INTRODUCTION", "2. IMPLEMENTATION" appear in the JSON output, and overall body parsing does not regress.
*   **Rollback Plan:** `git checkout extractor/semantic.py`.

### Milestone 2: Front Matter Geometry Experiment
*   **Objective:** Collect spatial telemetry for Front Matter objects.
*   **Code Change:** None to production. Write a new telemetry script.
*   **Benchmark:** Output telemetry for Spanner, BERT, Attention.
*   **Success Criterion:** Telemetry proves whether authors physically cluster or intersect specific abstract bounding boxes.
*   **Rollback Plan:** N/A (Script only).
